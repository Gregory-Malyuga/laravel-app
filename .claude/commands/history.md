# /history

Зафиксировать неожиданный сбой или паттерн в `docs/history/`.

## Шаги

1. Определить следующий номер: `HIST-<N>` (последняя запись в `docs/history/index.md` + 1)
2. Добавить запись в `docs/history/index.md` по шаблону ниже
3. Добавить строку-паттерн в `docs/history/checklist.md`

## Шаблон записи

```markdown
## HIST-00N — <краткое название>

**Дата:** YYYY-MM-DD
**Контекст:** Что делали, когда произошло.
**Симптом:** Что именно упало / повело себя неожиданно.
**Причина:** Почему это произошло.
**Фикс:** Что изменили для устранения.
**Паттерн для checklist:** <строка для docs/history/checklist.md>
```

## Строка в checklist

Добавить в таблицу `docs/history/checklist.md`:

```markdown
| <Паттерн в 3-5 словах> | [HIST-00N](index.md#hist-00n--краткое-название) |
```
