# Forbidden

Запрет-лист проекта. Дополняй по мере появления правил.

---

## Язык и типы

- **Запрещено `float` и `string` для денег** (если появится монетизация) — только `int` (копейки) или специализированный тип
- **Запрещено `mixed`** в сигнатурах методов — всегда уточняй тип
- **Запрещено `array` без аннотации** там, где структура известна

## Архитектура

- **Запрещены Eloquent-запросы вне `Infrastructure/Repositories/`** — Controller/Handler ходят в БД только через Repository
- **Запрещена бизнес-логика в Presentation** — контроллер только `commands->dispatch()` / `queries->ask()`
- **Запрещена бизнес-логика в Model** — модель тонкая (`$fillable`/`$casts`/отношения)
- **Запрещён импорт `Application`/`Infrastructure`/`Presentation` в `Domain`** — Domain изолирован (enforced deptrac)
- **Запрещён импорт чужого домена** (`Domains\Other\Application\...`) — связь через FK/id, события или `QueryBus`
- **`Command` и `Handler` — `readonly class`**; событие диспатчится из хендлера, не из модели/контроллера

## Отладка

- **Запрещено `dd()`, `dump()`, `var_dump()`, `print_r()` в коммите**
- **Запрещено `Log::debug()` с чувствительными данными** (пароли, токены, PII)

## Инфраструктура

- **Запрещено редактировать применённую миграцию** — только новая миграция
- **Запрещено коммитить `.env`** и файлы с секретами
- **Запрещено `--no-verify`** при коммите и пуше
